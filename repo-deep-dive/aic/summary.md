# AI for Industry Challenge (AIC) Toolkit: Summary

The **AI for Industry Challenge (AIC) Toolkit** is an open robotics competition platform developed by Intrinsic Innovation LLC that challenges participants to solve dexterous cable insertion in electronics assembly — a task currently bottlenecked by manual labor in manufacturing. The toolkit provides a complete simulation-to-evaluation infrastructure built on ROS 2 (targeting Kilted Kaiju), Gazebo, and standard robotics middleware, with a $180,000 prize pool for teams that progress through three phases culminating in real-world deployment at Intrinsic's HQ. Architecturally, it is split into an **evaluation component** (run by organizers: `aic_engine` for trial orchestration, `aic_bringup` for Gazebo simulation launch, `aic_controller` for impedance/force-managed robot control, and `aic_adapter` for sensor fusion) and a **participant model component** (what competitors submit: a ROS 2 Lifecycle node named `aic_model` that dynamically loads a Python `Policy` class at runtime and responds to the `/insert_cable` action with Cartesian or joint motion commands). Scoring is automated across three tiers — task success, insertion precision, and safety/efficiency — using custom Gazebo plugins (`ScoringPlugin`, `CablePlugin`) and the C++ `aic_scoring` package. What makes it distinctive is the clean separation between the evaluation harness and the contestant-facing `aic_model` framework, which reduces ROS 2 boilerplate to a single `insert_cable()` Python method, while still supporting advanced participants who implement their own node from scratch.
